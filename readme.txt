== Description ==

EMI SimpleCalc provides a clean, modern, and intuitive home loan EMI calculator for your WordPress site. Built with simplicity in mind, it allows users to instantly calculate their Equated Monthly Installment (EMI) using interactive sliders for Loan Amount, Interest Rate, and Loan Tenure.

The results are displayed in a visually appealing and easy-to-understand format, including a doughnut chart that breaks down the principal and total interest amounts. It is fully responsive and designed to be accessible to everyone, including audiences who are more comfortable with Hinglish.

**Features:**
* **Interactive Sliders & Inputs:** Easily adjust loan parameters.
* **Instant Calculation:** Results update in real-time with no page reloads.
* **Visual Doughnut Chart:** Clearly see the difference between principal and interest.
* **Full Amortization Schedule:** View a complete month-by-month payment breakdown with one click.
* **Minimalist & Modern Design:** A clean interface that fits any theme.
* **Fully Responsive:** Looks and works great on desktops, tablets, and mobile devices.
* **Lightweight:** Loads quickly and efficiently.

To use, simply place the `[home_loan_emi_calculator]` shortcode on any page or post.

== Installation ==

1.  Upload the `emi-simplecalc` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Add the shortcode `[home_loan_emi_calculator]` to any page, post, or widget where you want the calculator to appear.
